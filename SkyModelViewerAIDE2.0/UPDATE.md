# 更新日志

## v1.5.0 — 2026-07-15

### 新增

- **衣柜演示模式**：新增"可视化"/"衣柜演示"模式切换
  - 衣柜模式支持9个分类：裤子(body)、帽子(hat)、头发(hair)、面具(mask)、角(horn)、斗篷(wing)、颈部装饰(neck)、鞋子(feet)、背饰(prop)
  - 选择分类后自动筛选对应关键词的mesh模型
  - 每个分类各选一个模型，同时渲染在3D视口中
  - 切换某分类的模型只替换该分类，其他分类保持不变
  - 支持在衣柜模式下选择动画，所有部件共享同一套骨骼动画
  - 衣柜模式下顶部工具栏简化为：名称 + 动画按钮
  - 底部新增"清除"按钮，一键清空所有衣柜选择
- **多部件骨骼蒙皮**：MeshRenderer支持per-instance骨骼映射，每个部件独立骨骼权重但共享同一动画

### 修复

- **骨骼蒙皮矩阵修复**：`bone.matrix` 不再转置和求逆，直接作为IBM使用

## v1.4.0 — 2026-07-14

### 修复

- **骨骼蒙皮变形修复**：修复蒙皮矩阵转换错误导致"皮压细"和"乱面"的问题
  - 根因：`bone.matrix` 的平移分量存储在索引 `[12,13,14]`，与 `composeMat4` 产生的列主序格式一致，旧代码错误地对其做了转置，导致平移从 `[12,13,14]`（最后一列）跑到 `[3,7,11]`（底行），矩阵完全错乱
  - 修正：`bone.matrix` 本身即为逆绑定矩阵（IBM），直接复制使用，不再转置、不再求逆
  - `setAnimation()`：`bone.matrix` 从"转置+求逆"改为直接复制
  - `setMesh()`：mesh 文件骨骼矩阵从"转置"改为直接复制
  - 蒙皮公式：`B = W_anim * bone.matrix`，frame 0 时 mesh 从 T-pose 正确变形到动画第一帧

## v1.2.0 — 2026-07-12

### 新增

- **多格式导出**：模型支持导出 OBJ / GLB / FBX，贴图支持导出 PNG / JPEG
- **完整关卡导出**：地图导出包含地形(.meshes) + 所有放置的 mesh 实例（含位置变换矩阵）
  - 顶点位置应用 4x4 变换矩阵（旋转+平移）到世界空间
  - 法线应用旋转变换并重新归一化
  - UV 直接保留
  - 无地形的关卡也能导出（只要有 mesh 实例）
- **OBJ 导出器**：支持单模型、地形、完整关卡，按 group 分组
- **FBX 导出器**：ASCII FBX 7.4 格式，兼容 Blender/Maya/3ds Max
- **贴图导出器**：PNG 无损 / JPEG 有损（自动处理 Alpha 通道）
- **导出格式选择对话框**：根据选中资源类型（模型/贴图/地图）自动弹出对应格式选项

### 修复

- `packedAttributes` 字段名错误 → 修正为 `packedVertexAttrs`
- 加载完地图后导出按钮变灰 → 改为 `setEnabled(true)`
- 地图无地形数据时无法导出 → 现在只要有 mesh 实例就能导出

### 优化

- 导出成功后状态栏显示包含的 mesh 实例数量（如 `+305 meshes`）
- 关卡加载时保存所有 mesh 的 MeshData 和变换矩阵用于导出
- KTX 贴图加载时同步保存 Bitmap 用于 PNG/JPEG 导出
